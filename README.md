# Poneglyph

AI-powered document search and chat built on Gemini File Search for Retrieval-Augmented Generation (RAG).

Poneglyph helps you upload documents, create searchable File Search stores, and chat with an assistant that cites sources from your files.

## Highlights

- Secure authentication with NextAuth
- Upload PDFs, DOCX and text files with chunking and custom metadata
- Create and manage multiple File Search stores
- RAG chat that grounds replies in your documents and shows citations
- Conversation history per user with persistent storage via Prisma and PostgreSQL
- Built with Next.js (App Router), TypeScript and Tailwind CSS
